Meus Projetos

Calculadora de IMC

Este projeto é uma calculadora desenvolvida para auxiliar os usuários a descobrirem seu Índice de Massa Corporal (IMC). Com base na altura e no peso informados pelo usuário, o aplicativo calcula o IMC e indica a situação do usuário, ajudando a compreender melhor sua condição física.

Quiz Senac

Este projeto é um aplicativo de quiz desenvolvido em parceria com o Senac, criado em colaboração com colegas da faculdade. O app tem como objetivo ajudar os usuários a descobrir sua forma de aprendizado por meio de testes interativos, além de oferecer outras funcionalidades voltadas para o autoconhecimento educacional.

Adoção de Pets

Este projeto é um site desenvolvido para facilitar a adoção de pets. A plataforma permite o cadastro de clientes interessados em adotar, além do registro de animais disponíveis para adoção e de seus donos temporários. Os usuários podem visualizar informações sobre os pets, entrar em contato com os responsáveis atuais e concluir o processo de adoção de forma segura e eficiente..
